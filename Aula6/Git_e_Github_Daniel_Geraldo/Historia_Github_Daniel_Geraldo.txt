a. Quem inventou o sistema de controle de versão Git e por quê?
Por definição, o Git é um Sistema de Controle de Versão Distribuído. Ele foi criado por Linus Torvalds, que é mais conhecido por outra criação sua, o Linux.

b. Quem atualmente é o dono do Github e por quê?
Atualmente o dono do github é a microsoft após compra milionária da plataforma.

c. Existe outra maneira, além do terminal, de trabalhar com o Github?
Você pode criar arquivos e editar dentro do proprio github.
